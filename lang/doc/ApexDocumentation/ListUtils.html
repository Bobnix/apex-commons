sort

public static List<Object> sort(List<Object> objects, Comparator comp)
Sorts a list of objects based on the comparator passed in. Apex has a native sort for primatives, SObjects and Objects that implement the comparable interface but this gives much more controll by allowing a number of different comparators to be written and used.
Parameters
objects
The list to be sorted
comp
The comparator to sort with
Return Value
A new list with the sorted values.
